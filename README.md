# English

# FH6 Vinyl Tool for Linux (FVTL)

## Overview

**FH6 Vinyl Tool for Linux (FVTL)** is a Linux utility for importing and exporting **Forza Horizon 6** vinyl data while the game is running through Proton.

The project is currently in a testing phase, with the goal of providing a complete vinyl management and editing tool that runs entirely on Linux.

---

## Current Features

* Import vinyl data
* Export vinyl data
* Import JSON files generated by existing tools such as **ForzaPainter** and **KFPS**
* Runs natively on Linux

---

## Planned Features

* Vinyl data editor
* Generate vinyl data from images
* Vinyl management tools
* Performance improvements
* Additional editing features and quality-of-life improvements

## GPU Runtime Requirements

FVTL's geometrization feature uses **OpenCL** and some libraries also depend on **Vulkan**.

If the setup script does not install the required runtime libraries on your distribution, install them manually as shown below.

### Arch Linux / Manjaro

```bash
# ICD Loader
sudo pacman -S ocl-icd vulkan-icd-loader

# Install the package matching your GPU
sudo pacman -S intel-compute-runtime vulkan-intel   # Intel iGPU
sudo pacman -S rocm-opencl-runtime vulkan-radeon    # AMD (Mesa/RADV)
sudo pacman -S opencl-nvidia nvidia-utils           # NVIDIA
```

### Ubuntu / Debian

```bash
# ICD Loader
sudo apt install ocl-icd-libopencl1 libvulkan1

# Install the package matching your GPU
sudo apt install intel-opencl-icd mesa-vulkan-drivers   # Intel iGPU
sudo apt install mesa-opencl-icd mesa-vulkan-drivers    # AMD (Mesa/RADV)
sudo apt install nvidia-opencl-icd nvidia-driver-XXX    # NVIDIA (replace XXX with your driver version)
```

### Fedora

```bash
# ICD Loader
sudo dnf install ocl-icd vulkan-loader

# Install the package matching your GPU
sudo dnf install intel-opencl mesa-vulkan-drivers         # Intel iGPU
sudo dnf install mesa-libOpenCL mesa-vulkan-drivers       # AMD (Mesa/RADV)
sudo dnf install xorg-x11-drv-nvidia-cuda akmod-nvidia    # NVIDIA
```

### openSUSE

```bash
# ICD Loader
sudo zypper install ocl-icd libvulkan1

# Install the package matching your GPU
sudo zypper install intel-compute-runtime Mesa-libVulkan   # Intel iGPU
sudo zypper install Mesa-libOpenCL Mesa-libVulkan          # AMD (Mesa/RADV)
sudo zypper install nvidia-video-Gxx-kmp-default           # NVIDIA
```

## Verify the Installation

Check that your OpenCL and Vulkan runtimes are correctly installed.

### OpenCL

```bash
clinfo | head -20
```

> If `clinfo` is not installed, install the `clinfo` package provided by your distribution.

### Vulkan

```bash
vulkaninfo --summary
```

> If `vulkaninfo` is not installed, install the `vulkan-tools` package.

If you see **"Number of platforms"** and your GPU is listed, everything is working correctly.

---

## Troubleshooting

### `error while loading shared libraries: libOpenCL.so.1`

The OpenCL ICD loader is missing. Install the appropriate ICD loader package for your distribution.

### `error while loading shared libraries: libvulkan.so.1`

The Vulkan loader is missing.

> Even when using only the `-backend opencl` backend, some libraries are linked against Vulkan and require the Vulkan loader to be present.

### `cl: unknown error` / `no such device`

Your GPU's OpenCL runtime is missing or your graphics driver is outdated.

### `no Vulkan physical devices found`

Your Vulkan driver is missing, improperly installed, or the GPU is disabled (for example, in the system BIOS/UEFI).

## About the run script

When started, the script automatically:

* Activates the Python virtual environment (`.venv`)
* Temporarily adjusts `ptrace_scope` so FVTL can access the running FH6 process
* Launches FVTL

When FVTL exits, the script automatically:

* Deactivates the Python virtual environment
* Restores the original `ptrace_scope` value

Because of this, you normally do not need to activate `.venv` or modify `ptrace_scope` manually.

## About This Project

There are currently very few Linux tools capable of working with **Forza Horizon 6** vinyl data.

FVTL aims to provide a complete Linux-native environment for managing and editing vinyl data without requiring Windows.

The project is inspired by **KFPS** and **ForzaPainter** in terms of overall design and workflow.

However, to ensure proper license compliance and gain a deeper understanding of the implementation, every part of the source code has been written from scratch.

## トラブルシューティング

### `error while loading shared libraries: libOpenCL.so.1`

OpenCL ICDローダーがインストールされていません。

### `error while loading shared libraries: libvulkan.so.1`

Vulkanローダーがインストールされていません。

> `-backend opencl` のみ使用する場合でも、一部ライブラリのリンク依存として Vulkan ランタイムが必要です。

### `cl: unknown error` / `no such device`

GPUベンダー用のOpenCLランタイムがインストールされていない、またはGPUドライバーが古い可能性があります。

### `no Vulkan physical devices found`

Vulkanドライバーがインストールされていないか、GPUが無効化されている可能性があります（BIOS設定なども確認してください）。

### runスクリプトについて

起動スクリプトは以下を自動で実行します。

起動時

* Python仮想環境 (`.venv`) を有効化
* `ptrace_scope` を一時的に変更してFH6へアクセスできるよう設定
* FVTLを起動

終了時

* Python仮想環境を終了
* `ptrace_scope` を元の値へ復元

そのため、通常は手動で `.venv` を有効化したり `ptrace_scope` を変更する必要はありません。

---

### 設定タブ
* GUIの日本語と英語の切り替えが出来ます。
* 任意のバイナル保存フォルダを設定出来ます。

### 既存ツールのJSONのインポートの仕方
`vinyl/output/`辺りにコピペしたら読み込みます。

Linux向けには、Forza Horizon 6のバイナルデータを扱えるツールがほとんど存在しません。
